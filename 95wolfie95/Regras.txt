UMaResidencia

Projeto Colaborativo - Residências Universitárias

Uma das valências conferidas pela cadeira de Design Hipermédia é o Web Design, mais especificamente utilizando HTML5, CSS3 e Javascript.
Sendo assim, uma vez concluído o CTeSP - Tecnologias e Programação de Sistemas de Informação - é plausível o cenário em que tenha de realizar,
rapidamente, um protótipo de um website.

Em termos técnicos, pretende-se que sejam concretizados os seguintes items:

(5 valores) - Estrutura de navegação funcional (mínimo de 5 páginas);YES

(2 valores) - Tratamento de pelo menos 20 imagens a incluir numa galeria e/ou páginas do site;YES

(5 valores) - Inclusão de um elementos programados (Javascript, sass, etc);YES

(3 valores) - Inclusão de elementos originais e marcação semântica desses conteúdos;YES

(5 valores) - Design (CSS).OH YES

*Neste projeto não poderão ser usadas frameworks de CSS (como por exemplo o Bootstrap)
